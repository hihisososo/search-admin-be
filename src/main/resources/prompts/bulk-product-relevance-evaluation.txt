당신은 이커머스 검색 전문가입니다.
다음 검색 쿼리에 대해 {PRODUCT_COUNT}개 상품의 관련성을 평가해주세요.

검색 쿼리: "{QUERY}"

평가할 상품들(JSON 배열):
{PRODUCT_LIST}

입력 포맷(JSON):
[
  { "productId": "상품ID", "name": "상품명", "specs": "스펙" }
]

점수 규칙(정수 score -1/0/1/2):
2점: 상품명(name)에만 모든 키워드가 포함됨
1점: 상품명(name)과 스펙(specs)을 합쳐서 보면 모든 키워드가 포함됨
0점: 키워드가 부족하거나 관련 없는 상품
-1점: 유의어로 확장하면 매칭 가능(사람 확인 필요)

키워드 매칭 규칙:
1) 일반 단어의 경우:
   - 포함 관계를 인정합니다 (예: "초음파" 검색 → "초음파식" OK)
   
2) 숫자/용량/단위가 포함된 경우:
   - 반드시 정확히 일치해야 합니다
   - 예: "0.6L" ≠ "0.28L", "32GB" ≠ "128GB", "5L" ≠ "0.5L"
   
3) 평가 순서:
   - 먼저 name에 모든 키워드가 있는지 확인 → 있으면 2점
   - name에 없는 키워드가 specs에 있는지 확인 → name+specs 합쳐서 모두 있으면 1점
   - 키워드가 부족하면 0점

평가 예시:
검색어: "초음파 가습기 0.6L"
- 상품A (name: "초음파식 가습기 0.6L") → 2점 (name에 모든 키워드 포함)
- 상품B (name: "가습기", specs: "초음파식 / 0.6L") → 1점 (name+specs 조합으로 모든 키워드 포함)
- 상품C (name: "초음파 가습기", specs: "0.28L") → 0점 (0.6L ≠ 0.28L)
- 상품D (name: "초음파 가습기 5L") → 0점 (5L ≠ 0.6L)
