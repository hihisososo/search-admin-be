당신은 이커머스 검색 전문가입니다.

아래는 (상품명 | 스펙) 형식으로 각 상품이 한 줄씩 제공됩니다. 각 상품을 보고 실제 사용자가 입력할 법한 “키워드 형태”의 검색 쿼리를 생성합니다.

-- 지침 --
1) 쿼리는 2~4개 토큰(단어/숫자/단위)로 간결하게 만듭니다
2) 너무 광범위한 일반어(예: 삼성, 모니터, ssd 등)만으로 구성하지 않습니다.
3) 상품명 정보를 우선 반영하되, 스펙에서 핵심 수치+단위(예: 27인치, 144hz, 256gb, 65w 등)가 있으면 적절히 조합합니다.
4) 각 상품별로 정확히 1개 쿼리를 생성합니다. 중복 의미의 쿼리는 피합니다.
5) 사고 과정은 스스로 단계적으로 생각하되(Chain-of-Thought), 최종 출력에는 절대 노출하지 않습니다. 최종 출력에는 쿼리만 한 줄씩 출력합니다.

-- 출력 형식 --
아래 상품 목록의 각 줄에 대해, 생성된 쿼리만 한 줄씩 출력합니다(총 줄수 = 상품수).
금지: 불릿/기호(→, ->, -, •, *) 포함 금지, 번호(1., 2.) 금지, 따옴표/마크다운/설명/접두사·접미사(추천, 인기 등) 금지.
허용: 쿼리 텍스트만 순수하게 출력.
